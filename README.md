Recursividade em Java – Contagem de Dígitos de um Número Inteiro

Enunciado
Criar uma aplicação em Java com uma função recursiva que receba um **número inteiro** como parâmetro e retorne a **quantidade de dígitos** desse número.

A função deve:
* Identificar quando o número possui **apenas 1 dígito** (condição de parada).
* Reduzir o número progressivamente dividindo por 10, contando os dígitos restantes até a condição de parada ser alcançada.
* Ser capaz de lidar com números **positivos, negativos e zero**.


Objetivo

O objetivo deste exercício é praticar:
* Recursividade em Java
* Identificação e implementação da **condição de parada**
* Redução do problema em termos do **termo anterior**
* Raciocínio lógico aplicado à manipulação de números inteiros
* Compreensão da relação entre cada chamada recursiva e a contagem de dígitos


Requisitos

O programa deve:
1. Receber um número inteiro como entrada.
2. Implementar a contagem de dígitos utilizando **apenas recursão**, sem laços de repetição (`for`, `while`).
3. Identificar corretamente a **condição de parada**: quando o número tiver apenas um dígito.
4. Garantir que a função retorne o número correto de dígitos em todos os casos.
5. Testar a aplicação com exemplos variados, incluindo números de 1 dígito, múltiplos dígitos e valores extremos do tipo `int`.


Tecnologias
* Oracle Java
* Recursividade
* Manipulação de números inteiros


Observações

Este exercício faz parte dos estudos de **Introdução às Estruturas de Dados**, com foco no desenvolvimento do raciocínio lógico, na compreensão de técnicas recursivas e na redução gradual do problema.
